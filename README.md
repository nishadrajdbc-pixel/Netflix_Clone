# Netflix Clone 🎬

A responsive Netflix Landing Page Clone built using **HTML, CSS, and JavaScript**.

This project recreates the modern Netflix UI with a hero banner, responsive navigation bar, horizontal movie carousels, search functionality, and a clean footer.

---

## 🚀 Features

* Responsive Netflix-inspired UI
* Fully functional navigation bar
* Mobile-friendly hamburger menu
* Hero banner section
* Auto-changing featured content slider
* Horizontal movie carousels
* Smooth scrolling carousel controls
* Movie search functionality
* Netflix-style hover animations
* Responsive design for desktop, tablet, and mobile devices
* Footer section

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**

## ⚙️ Installation & Usage

### 1. Clone the repository

```bash
git clone https://github.com/your-username/netflix-clone.git
```

### 2. Navigate into the project folder

```bash
cd netflix-clone
```

### 3. Open the project

Simply open:

```text
index.html
```

in your browser.

You can also use **VS Code Live Server** for a better development experience.
